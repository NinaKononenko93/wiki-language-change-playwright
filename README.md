# Playwright E2E тест: Зміна мови інтерфейсу Wikipedia

##  Опис проєкту

Цей проєкт містить автоматизований end-to-end тест для перевірки функціональності зміни мови інтерфейсу для авторизованого користувача Wikipedia.

Сценарій покриває:
- авторизацію користувача
- перехід до налаштувань (Preferences)
- зміну мови інтерфейсу
- перевірку застосування змін

---

##  Тест-кейс

**Назва:** Зміна мови інтерфейсу для авторизованого користувача

**Кроки:**
1. Авторизуватися в системі
2. Перейти в розділ Preferences
3. Відкрити вкладку "User profile"
4. Обрати іншу мову інтерфейсу
5. Зберегти зміни
6. Перезавантажити сторінку
7. Перевірити, що мова змінилась

**Очікуваний результат:**
Інтерфейс відображається обраною користувачем мовою.

---

##  Використані технології

- Playwright (TypeScript)
- Node.js
- Docker
- Page Object Model (POM)

---

## Як запустити

###  Локально

```bash
npm install
npx playwright test
```

---

###  У Docker

**Збірка образу:**

docker build -t wiki-playwright .

**Запуск контейнера:**

```bash
docker run --rm --env-file .env -v ${PWD}/playwright-report:/playwright-test/playwright-report wiki-playwright
```

---

### Звіт про тестування
Після виконання тесту HTML-звіт доступний у папці:

```bash
playwright-report
```

---

###  Конфігурація

Чутливі дані (логін/пароль) зберігаються у .env файлі та не додаються до репозиторію.
